# Newsletter

## Opcion A: Google Form

Si el admin configuro `NEWSLETTER_FORM_URL`, veras un boton de suscripcion en cada post.

## Opcion B: fallback gratuito

Si no hay formulario, el sitio usa un enlace `mailto:` y te abre tu cliente de correo con asunto predefinido.

Tambien puedes contactar por el email configurado en `NEWSLETTER_EMAIL`.
